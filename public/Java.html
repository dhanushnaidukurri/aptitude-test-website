Core Java

  • Core Java refers to the fundamental features and components of the Java programming language

  • This subtopic includes basic concepts like variables, data types, control structures, and object-oriented programming (OOP) principles

  • It also covers the Java Standard Library (java.lang)

  • Foundational classes for input/output operations, exception handling, and basic utility functions

Java API's

  • The core set of libraries and classes for essential functions like collections (java.util), file handling (java.io), and networking (java.net)

  • APIs for building enterprise-level applications, including Servlets, JSP (JavaServer Pages), EJB (Enterprise JavaBeans), and JPA (Java Persistence API)

  • A rich client platform for building desktop applications with GUIs

  • An API for developing applications for mobile and embedded systems

  • Libraries for implementing web services using technologies like JAX-RS and JAX-WS

JVM

  • JVM works and its components like class loader, bytecode verifier, and execution engine

  • JVM manages memory, including the heap, stack, and method area

  • Techniques and algorithms used by the JVM to manage memory and clean up unreferenced objects

  • Optimization and configuration of the JVM for performance improvements

Java Frame Works

  • A comprehensive framework for building enterprise applications, known for dependency injection and aspect-oriented programming

  • An Object-Relational Mapping (ORM) framework for simplifying database interactions

  • A framework for building web applications based on the Model-View-Controller (MVC) pattern

  • A reactive web application framework for building scalable and high-performance applications

Application Domains

  • Java is commonly used for web applications, with technologies like Servlets, JSP, and web frameworks

  • Java is used for Android app development

  • Java is popular for building large-scale enterprise applications and server-side software

  • Java is used in scientific computing and embedded systems development

  • Java plays a role in big data technologies like Hadoop and in building cloud-native applications
