OOP

  • Classes: Understand how to define classes in C++ to encapsulate data and behavior

  • Objects: Learn how to create objects from classes and access their member variables and functions

  • Inheritance: Explore the concept of inheritance, allowing classes to inherit properties and behaviors from other classes

  • Polymorphism: Understand how polymorphism enables objects of different classes to be treated as objects of a common base class

  • Encapsulation: Discover how encapsulation helps in hiding the internal implementation details of a class and provides data abstraction

STL

  • Containers: Learn about various container classes provided by the STL, such as vectors, lists, queues, and stacks

  • Algorithms: Understand the different algorithms available in the STL, including sorting, searching, and manipulating elements in containers

  • Iterators: Explore the concept of iterators, which provide a way to traverse and access elements in containers

  • Function Objects: Discover how function objects (functors) can be used with algorithms to customize their behavior

  • Generic Programming: Understand how the STL promotes generic programming by providing reusable components that work with different data types

Memory Management

  • Stack and Heap Memory: Learn about the stack and heap memory regions and how they are used for storing variables and objects

  • Dynamic Memory Allocation: Understand how to allocate memory dynamically using the new operator and deallocate it using the delete operator

  • Memory Leaks: Discover the importance of proper memory deallocation to avoid memory leaks and resource wastage

  • Smart Pointers: Explore the use of smart pointers, such as unique_ptr and shared_ptr, to automate memory management and prevent memory leaks

  • RAII (Resource Acquisition Is Initialization): Understand the RAII principle and how it can be used to manage resources, including memory, in C++

Exception Handling

  • Try-Catch Blocks: Learn how to use try-catch blocks to handle exceptions and gracefully recover from errors

  • Throwing Exceptions: Understand how to throw exceptions to indicate exceptional conditions in your code

  • Exception Classes: Explore the concept of creating custom exception classes to provide more specific error information

  • Exception Safety: Discover techniques for writing exception-safe code to ensure proper resource cleanup and prevent memory leaks

  • Exception Specifications: Understand the use of exception specifications to declare the types of exceptions that a function can throw

File Handling

  • File Streams: Learn how to work with file streams in C++ to read from and write to files

  • Opening and Closing Files: Understand the process of opening and closing files using file stream objects

  • Reading and Writing Data: Explore techniques for reading and writing different types of data to and from files

  • Error Handling: Learn how to handle errors and exceptions that may occur during file operations

  • File Manipulation: Discover additional file manipulation operations, such as renaming, deleting, and seeking within files
