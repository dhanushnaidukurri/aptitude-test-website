About :

  • Processes are independent execution units with their own memory and resources.

  • The OS scheduler decides which process to execute next.

  • Processes can communicate and synchronize using inter-process communication (IPC) mechanisms

  • Multithreading allows for multiple threads within a single process

  • Process management is critical for resource allocation and system stability

About :

  • Virtual memory provides an abstraction of physical RAM, allowing efficient memory allocation.

  • Paging and segmentation are common memory allocation techniques

  • Memory protection prevents one process from accessing the memory of another

  • The memory hierarchy includes RAM, disk storage, and cache levels.

  • Memory leaks and fragmentation can lead to performance issues.

  • Security and access control mechanisms are essential for file system management.

  • Journaling and backup systems ensure data integrity and recovery.
