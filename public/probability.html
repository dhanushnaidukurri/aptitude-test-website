Formulas on Conditional Probability :

  • Conditional Probability Formula: P(A|B) = P(A and B) / P(B)

  • Multiplication Rule for Conditional Probability: P(A and B) = P(B) * P(A|B)

  • The fraction a/b represents a equal parts out of a total of b equal parts.

  • Bayes' Theorem: P(A|B) = [P(B|A) * P(A)] / P(B)

  • Law of Total Probability: P(A) = Σ [P(A|B) * P(B)] (sum over all possible events B that partition the sample space)

  • Independence of Events: If A and B are independent, then P(A and B) = P(A) * P(B)

Formulas on Probability Distributions:

  • Binomial Probability Distribution: P(X = k) = C(n, k) * p^k * (1-p)^(n-k), where X is a binomial random variable, n is the number of trials, k is the number of successful outcomes, and p is the probability of success.

  • Normal Distribution Probability Density Function: f(x) = (1 / (σ√(2π))) * e^(-((x-μ)^2 / (2σ^2))), where μ is the mean and σ is the standard deviation.

  • Poisson Probability Mass Function: P(X = k) = (e^(-λ) * λ^k) / k!, where X is a Poisson-distributed random variable, and λ is the average rate of occurrence.

  • Exponential Probability Density Function: f(x) = λ * e^(-λx), where X is exponentially distributed with a rate parameter λ.

  • Uniform Distribution Probability Density Function: f(x) = 1 / (b - a) for a ≤ x ≤ b, where X is uniformly distributed between a and b.

Formulas on Bayesian Probability :

  • Prior Probability: P(A), the initial probability of an event A

  • Likelihood: P(B|A), the probability of observing evidence B given that A is true.

  • Posterior Probability: P(A|B), the updated probability of A given the evidence B, calculated using Bayes' Theorem.

  • Law of Total Probability (Extension): P(A) = Σ [P(A|B) * P(B)], where the sum is taken over all possible events B, even when A and B are conditional probabilities themselves.

  • Maximum A Posteriori (MAP) Estimation: Finding the value of A that maximizes P(A|B) in Bayesian inference.

Markov Chains

  • Transition Probability Matrix: A square matrix P, where P[i][j] represents the probability of transitioning from state i to state j.

  • Chapman-Kolmogorov Equation: P^(n) = P(1) * P(2) * ... * P(n), which describes the probability of transitioning from state i to state j in n steps.

  • Stationary Distribution: π = πP, where π is the stationary distribution vector, and P is the transition probability matrix.

  • First-step Analysis: Probability of reaching a particular state after the first step in a Markov chain, P(X = j | X(0) = i) = P(i, j).

  • Markov Chain Absorption Probability: The probability of being absorbed into a specific state starting from a certain state in a absorbing Markov chain.

Formulas on Random Variables

  • Expected Value (Mean) of a Discrete Random Variable (X): E(X) = Σ [x * P(X=x)]

  • Variance of a Random Variable (X): Var(X) = E[(X - μ)^2], where μ is the mean of X.

  • Expected Value of a Function of a Random Variable: E[g(X)] = Σ [g(x) * P(X=x)]

  • Linearity of Expectation : E(aX + bY) = aE(X) + bE(Y) for constants a and b.

  • Variance of a Sum of Random Variables: Var(X + Y) = Var(X) + Var(Y) if X and Y are independent
