Formulas on Set Theory :

  • Union of Sets: A ∪ B = {x | x ∈ A or x ∈ B}

  • Intersection of Sets: A ∩ B = {x | x ∈ A and x ∈ B}

  • Complement of a Set: A' (read as "A complement") = {x | x is not in A}

  • Set Difference: A - B = {x | x ∈ A and x ∉ B

  • De Morgan's Laws: (A ∪ B)' = A' ∩ B'
        (A ∩ B)' = A' ∪ B'

Formulas on Combinatorics:

  • Permutations of n objects taken r at a time: P(n, r) = n! / (n - r)!

  • Combinations of n objects taken r at a time: C(n, r) = n! / (r!(n - r)!)

  • Binomial Coefficient: C(n, r) = n! / (r!(n - r)!)

  • Pigeonhole Principle: If n items are placed into m containers, and n > m, then at least one container must contain more than one item.

  • Inclusion-Exclusion Principle: |A ∪ B| = |A| + |B| - |A ∩ B|

Formulas on Graph Theory:

  • Euler's Formula for Planar Graphs: For a connected planar graph with V vertices, E edges, and F faces: V - E + F = 2.

  • Handshaking Lemma: In any graph, the sum of the degrees of all vertices is equal to twice the number of edges.

  • Distance in a Graph: The distance between two vertices u and v in a graph is the length of the shortest path between them.

  • Tree Spanning: In a connected graph, a tree is a connected acyclic subgraph that spans all vertices.

  • Dijkstra's Algorithm: An algorithm for finding the shortest path from a source vertex to all other vertices in a weighted graph.

Number Theory:

  • Prime Factorization Theorem: Every integer greater than 1 can be expressed as a unique product of prime numbers (fundamental theorem of arithmetic).

  • Euclidean Algorithm: A method for finding the greatest common divisor (GCD) of two integers.

  • Modular Arithmetic: a ≡ b (mod m) if and only if m divides (a - b).

  • Fermat's Little Theorem: If p is a prime number and a is not divisible by p, then a^(p-1)≡ 1 (mod p).

  • Chinese Remainder Theorem: A theorem that provides a solution to a system of simultaneous modular congruences.

Formulas onDiscrete Structures:

  • Function Composition: (f ∘ g)(x) = f(g(x)), where ∘ represents function composition.

  • Relations: A relation R on a set A is reflexive if (a, a) ∈ R for all a in A, symmetric if (a, b) ∈ R implies (b, a) ∈ R, and transitive if (a, b) ∈ R and (b, c) ∈ R implies (a, c) ∈ R.

  • Recurrence Relation: An equation that relates a term of a sequence to previous terms in the sequence.

  • Generating Function: A power series used to represent a sequence of numbers, often used in combinatorics.

  • Principle of Inclusion-Exclusion: A technique to count the elements in a union of sets while correcting for overcounting.
